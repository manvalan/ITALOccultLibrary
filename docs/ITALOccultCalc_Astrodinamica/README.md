## Italiano

### Descrizione

Questo manuale scientifico bilingue è stato creato per il progetto **ITALOccultLibrary**. Fornisce una trattazione completa e rigorosa dei fondamenti matematici e fisici dell'astrodinamica, con particolare attenzione a:

- **Sistemi di Coordinate**: Principi fondamentali dei sistemi di riferimento astronomici
- **Piani di Riferimento**: Trasformazioni tra diversi sistemi di riferimento
- **Scale Temporali**: JD, MJD, UTC e altri standard temporali astronomici

Ogni capitolo include spiegazioni teoriche, algoritmi con formule, schemi illustrativi e esempi pratici con dati reali.

## English

### Description

This bilingual scientific manual was created for the **ITALOccultLibrary** project. It provides a complete and rigorous treatment of the mathematical and physical foundations of astrodynamics, with particular attention to:

- **Coordinate Systems**: Fundamental principles of astronomical reference systems
- **Reference Planes**: Transformations between different reference frames
- **Time Scales**: JD, MJD, UTC and other astronomical time standards

Each chapter includes theoretical explanations, algorithms with formulas, illustrative diagrams, and practical examples with real data.
